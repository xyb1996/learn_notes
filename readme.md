## 推荐系统介绍

## 推荐系统的主要元素
物品集合、用户、场景、推荐引擎、推荐结果集。

## 推荐引擎重要模块
* 召回模块：通过多种召回方法进行组合召回，挑选用户最可能感兴趣的物品，比如：用户最近点击物品召回相似物品，用户兴趣类目召回物品。
* 排序模块：针对召回模块对物品进行精排，常用特征有：标签特征、物品的特征以及组合特征。
* 后排模块：需要对排序列表进行调整，如运营干预、优先级调权、指定下发规则。

# 第二章推荐系统中的召回算法

## 召回方法主要类别
* 行为相似召回：通过如购买行为、点击行为、评论行为，得到该行为指向行为的相似物品，常见的有协同过滤算法。
* 相似用户召回；通过用户画像和用户行为，计算相似用户，根据最近邻算法来得到推荐结果。
* 内容相似召回：通过对***物品内容的***理解和分析，得到物品相似度，根据用户对该物品的行为，得到推荐结果，常见的有Word2vec算法。

## 协同过滤概述
（1）收集用户偏好（2）找到相似的用户或物品（3）计算并推荐
